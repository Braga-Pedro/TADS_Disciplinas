# Requisitos Calculadora POO
O objetivo é criar uma calculadora utilizando o paradigma POO, com métodos para executar ações característica do objeto

### Atributos
- operando 1 : float
- operando 2 : float
- valor : float

### Métodos
- Soma
- Subtração
- Divisão
- pegaOperandos
  Scan para pegar o valor do teclado
